박스모델 2 (Box-Model)


박스 모델 중심에는 컨텐츠 영역이 있음.
width(위스) height(하이트) 를 각각 100px씩 주면 너비와 높이가 각 100px인 정사각형이 만들어짐
이렇게 만들어진 박스 실제 요소의 크기에는 테두리 까지 포함되서 계산됨.
padding(페딩)과 margin(마진) 역시 크기가 추가됨



Ex

margin의 크기 + padding의 크기까지 요소의 크기에 잡힘 (border역시 잡힘)

이에 대한 대응책으로 box-sizing을 씀
box-sizing
속성 값 처리 값
content-box 기본값 / 너비와 높이가 콘텐츠 영역만을 포함
border-box height와 width가 padding과 border까지 포함
=width 와 height 가 같더라도, box-sizing 속성값에 따라 크기가 달라질 수 있다

box-sizing: border-box; 을 넣으면 그림과 같이 100x100 으로 맞춰짐
    내용 정리
  1. 박스 모델에 너비와 높이를 지정하면, 콘텐츠 영역의 크기가 결정
  2. box-sizing속성은 너비와 높이가 포함할 영역을 변경함으로써 너비와 높이
    의 계산 방법을 결정할 수 있다.
  3. box-sizing의 기본값은 content-box이다
  4. box-sizing을 border-box로 지정하면 테두리까지 너비와 높이에 포함된다

요소의 배경색상이나 배경 이미지

    background
  • 배경(background)은 콘텐츠의 배경을 정의
    단축 속성으로써 색상,이미지,반복, 등 다양한 하위 속성을 정의할 수 있음

    • 하위 속성 역활
    background-color 배경 색을 정의
    background-image 배경 이미지를 정의
    background-position 배경 이미지의 초기 위치를 정의
    background-size 배경 이미지의 크기를 정의
    background-repest 배경 이미지의 반복 방법을 정의

  • background-color )

    속성값을 지정할때 color과 같음
  • background-image )

    url();함수를 사용하여 이미지 *기본값
    사진의 크기보다 div박스가 더 커서 부족한 부분을 반복적으로 채워짐. (기본값)
    이 기본값을 바꾸고싶으면 리핏(repest)을 이용
  • background-repest )

    backgrund-repest 의 기본값은 repeat(반복한다)이다.
    반복하지 않게 만들어 주고싶으면 속성값을 no-repest로 바꿔주면 반복되지않음
  • background-position )

    left , right , center , top , bottom 사용가능
    말 그대로 위치를 지정하는 속성.
    background-position의 기본값은 left top;
  • background-size )

    cover , contain
    말 그대로 사이즈를 조절 background-size: 가로px 세로px;
    키워드 값도 지정가능 대표적인것 두가지는
    cover; 이미지가 찌그러지지 않는 한도 내에서 최대치로 지정
    contain; 이미지가 찌그러지거나 잘리지않는 한도 내에서 최대치로 지정
  • 이렇게도 사용 가능

    *순서는 상관없음
    내용 정리
  1. background는 콘텐츠의 배경을 정의.
  2. 단축 속성으로써, 다양한 하위 속성을 포함
  3. 하위 속성을 한번에 정의할 경우, 정의할 속성과 정의를 생략할 속성을
    다양하게 석택할 수 있어 여러 경우의 수가 존재함
    4. Tip! 정의 시 여러 경우의 수가 존재하는 속성을 외우려고 하기보다는 계속 해보면서 체득하는 방향으로 공부하면 좋음
