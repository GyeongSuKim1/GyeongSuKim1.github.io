flexbox 다루기

flexbox를 다루기 위해 아래와 같은 속성들을 사용할 수 있다.

  • 주축 배치 방법 : justify-content
  • 교차축 배치 방법 : align-items
  • 교차축 개별 요소 배치 방법 : align-self
  • 줄 바꿈 여부 : flex-weap
    • 아래 상세 설명
  • 주축 배치 방법 : justify-content
    하위 속성
    속성값 의미
    flex-start 플렉스 컨테이너의 앞쪽에다 요소를 배치
    flex-end 플렉스 컨테이너의 끝에다 요소를 배치
    center 플렉스 컨테이너의 가운데다 요소를 배치
    space-around 주축에서 요소들 간의 여백을 동일하게 맞춤
    space-between 첫번째 요소와 마지막 요소를 양 끝에 붙이고 나머지 요소들간의 공간을 동일하게 함
    • 정리 하자면 저스티파이 컨텐츠는 주축에서의 요소들간의 배치 간격을 결정한다

    순서대로 flex-start, flex-end, center, space-around, space-between
  • 교차축 배치 방법 : align-items
    • 주축을 justify-content로 만들었으니 교차축은 align-items태그로 만듬

    순서대로 flex-start, flex-end, center
  • 교차축 개별 요소 배치 방법 : align-self
    • 교차축에서 요소 하나의 배치방법을 결정하는 속성

    Ex
    1
    2
    3
    한가지 요소만 지정해서 교차축 배치를 결정 할 수 있다
  • 줄 바꿈 여부 : flex-weap
    • 플렉스랩의 역활은 플렉스컨테이너의 하위 요소들이
    플렉스 컨테이너보다 커져버렸을때의 처리방법을 결정하는 속성

    하위 속성
    속성값 의미
    nowrap 기본값,
    wrap 주축으로 진행되던 하위요소들의 도합 크기가 플렉스 컨테이너의 축 길이보다 커졌을때 이것을 두행이상으로 처리함
    wrap-reverse wrap의 반대방향으로 하겠다.
    내용정리
  1. justify-content는 flexbox 주축에서의 요소 배치 방법을 정의한다.
  2. align-items는 flexbox교차축에서의 요소 배치 방법을 정의한다.
  3. align-self는 flexbox내 개별 요소의 교차축 배치 방법을 정의한다.
  4. flex-wrap는 개별 요소들의 도합 크기가 컨테이너 주축 길이보다
    커졌을 때의 처리 방법 및 방향을 정의한다.
