### (백준/C++) 1110번 더하기 사이클
#### 📌 제출번호: 38732141
``` cpp
#include <iostream>
using namespace std;

int main() {
    int times = 0;
    int num, init, sum;
    int a, b, c;

    cin >> num;

    init = num;

    while (1) {
        a = num / 10; // 2 6 8 4
        b = num % 10; // 6 8 4 2
        sum = a + b; // 8 14 12 6

        num = (b * 10) + (sum % 10); // 68 84 42 26

        times++;

        if (num == init)
            break;
    }

    cout << times << "\n";

    return 0;
}

```

#### 📌 풀이
1. 26 -> 2 + 6 = 8 -> 68
2. 68 -> 6 + 8 = 14 -> 84
3. 84 -> 8 + 4 = 12 -> 42
4. 42 -> 2 + 6 = 8 -> 68
