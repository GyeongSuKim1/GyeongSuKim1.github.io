테이블<table>표 만들기

  • <table> : 테이블을 생성하는 태그
  • <th> : 테이블의 헤드부분 (자동으로 가운데 정렬, 굵게 적용)
  • <tr> : 테이블의 행(가로 한줄)을 담당
  • <td> : 테이블의 열 담당
  • <caption> : 테이블에 캡션을 붙일때 사용<table> 태그 다음에 바로 사용
  • <thead> : 테이블의 헤드 영역 지정
  • <tbody> : 테이블의 바디 영역 지정

위에 태그를 사용해서 표를 만들면
시작은<table></table>로 시작과 바깥을 감싸고
행을 만드는<tr></tr>태그를 사용해서 그안에
열을 만드는<td></td>태그를 삽입해줘야함

맨 위에 <caption>생선 가격</capion>으로 제목을 설정해준다음
<thead></thead>그 다음 <tr></tr>로 행을 생선해주고
<th></th>테이블의 헤드를 만들어줌
이 다음부터는 <td></td>열을 만들어주면 됨

간단하게 설명하면 테이블 태그를 삽입하고
캡션으로 제목을 설정후
tr태그로 열을 만든후 그 안에 td태그를 넣어주면 끝

그 외 속성들
속성 설명
border 테이블의 테두리(두께)를 지정
bordercolor 테이블의 테두리 색상
bgcolor 배경색을 지정
align 정렬을 지정(left,center,right)
cellspacing 셀간의 간격을 지정
width 가로크기 지정
height 세로크기 지정
colspan 열 병합(가로)
rawspan 행 병합(세로)
