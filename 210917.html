CSS

HTML 문서를 아름답게 꾸며주는 CSS
HTMl + CSS = 예쁜,멋진,깔끔한,화려한 웹페이지
(HTML의 용도,문법 등 HTML에 대한 기본적인 지식이 필요)

CSS란?

CSS는 Cascading Style Sheets를 의미함

  • Cascading : 계단식
  • Style : 멋을 내다
  • Sheets : (종이)한 장
    • =계단식으로 스타일을 정의하는 문서
CSS는 HTMl문서에 스타일을 더해줌
HTML문서는 태그가 태그를 포함하는 계단식 구조를 지니고 있다

CSS가 일할려면?

CSS문서(코드) 자체는 독립적으로 존재할 수 있지만, 그 목적이 구조화된 HTML문서의
스타일을 정의하는 데 있으므로 HTML문서가 없는 CSS문서는 사실상 의미 없음

  • HTML : 웹 문서의 콘텐츠를 구성하는 언어
  • CSS : 웹 문서의 콘텐츠에 스타일을 추가하는 언어(색상,크기,위치 등등)
    • = HTML과 함께 사용되지 않는 CSS는 단순 텍스트일 뿐, 웹브라우저를 통해 결과물을
    확인하기 위해서는 반드시 HTML문서 작성이 선행되어야함 !!!!!

    내용정리

  1. CSS는 웹문서를 꾸며주기 위해 사용하는 언어
  2. CSS코드는 CSS파일에 작성한다.확장자는~.CSS이다
  3. CSS코드는 HTML문서에 스타일을 추가(정의)하는 언어이다
    4. 따라서 HTML문서가 존재하지 않을 때는 단순 텍스트에 불과하다
